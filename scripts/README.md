# 模拟数据脚本

本目录包含直播竞拍系统的模拟数据生成脚本。

## 脚本说明

- `seed-data.ts`: 生成模拟数据的主脚本，包括用户、直播间和商品数据

## 使用方法

### 从 backend 目录运行（推荐）

直接在 `backend/` 目录下运行脚本，这样可以复用已有的环境配置和依赖：

```bash
cd ../backend
npm run db:seed
```

### 在 scripts 目录独立运行（备选）

如需在 `scripts/` 目录下独立运行：

1. 配置环境变量
   ```bash
   cp ../backend/.env.example .env
   # 编辑 .env 文件，填入数据库连接信息
   ```

2. 创建 Prisma schema 链接
   ```bash
   ln -sf ../backend/prisma .
   ```

3. 安装依赖
   ```bash
   npm install
   ```

4. 运行脚本
   ```bash
   npm run seed
   ```

## 生成的数据

脚本将生成以下数据：

- **20 个用户**: 每个用户有唯一的手机号、抖音ID、头像和昵称
- **20 个直播间**: 每个用户对应一个直播间
- **100 个商品**: 每个直播间有 5 个商品

## 默认密码

所有生成的用户密码均为：`123456`

## 注意事项

- 脚本在运行前会清空现有数据，请谨慎使用
- 确保数据库连接正常
- 如需要调整数据量，可以修改脚本中的对应参数
