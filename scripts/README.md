# 模拟数据脚本

**注意**：模拟数据脚本已移至 `backend/src/seed.ts`，请在 backend 目录下运行。

## 使用方法

在 `backend/` 目录下运行脚本：

```bash
cd backend

# 确保 Prisma Client 已生成（如果没生成过）
npm run prisma:generate

# 运行模拟数据脚本
npm run db:seed
```

## 生成的数据

脚本将生成以下数据：

- **20 个用户**: 每个用户有唯一的手机号、抖音ID、头像和昵称
- **20 个直播间**: 每个用户对应一个直播间
- **100 个商品**: 每个直播间有 5 个商品

## 默认密码

所有生成的用户密码均为：`123456`

## 注意事项

- 脚本在运行前会清空现有数据，请谨慎使用
- 确保数据库连接正常
- 如需要调整数据量，可以修改 `backend/src/seed.ts` 中的对应参数
